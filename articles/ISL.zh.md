# 冰岛：地热能源底盘与危机后重塑的小型开放经济体

冰岛最适合被映射为：具有极强辨识度的**地热能源底盘**（并叠加水电）的**小型开放经济体**。同时，2008年的银行体系崩溃是其现代宏观金融叙事的关键转折点，之后冰岛重建金融体系与政策框架，形成"危机后治理与韧性"的国家标签。

## 关键事实与结构特色

- **地热优势显著：** 冰岛以地热发电与地热供暖的高渗透率闻名，是其最核心的结构标签。
- **2008危机转折：** 2008年发生重大金融危机与银行体系崩溃，深刻塑造其当代金融结构与政策框架。
- **小型开放经济逻辑：** 经济对外部需求（旅游与出口）与外部金融条件相对敏感。

## Core Pillars（金融层）

**地热能源底盘 + 危机后宏观金融重塑**
以能源自给与地热优势支撑经济底盘，金融体系在2008危机后重建并强化政策韧性。

## Global Identity（直观层）

**地热岛屿经济体**
以地热能源与危机后小国宏观叙事被全球识别。
