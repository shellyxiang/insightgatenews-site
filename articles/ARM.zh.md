# 亚美尼亚：侨民链接的服务型经济（通道受限）

亚美尼亚不是能源出口国，更适合写成：**侨民链接 + 服务/能力建设型**的小型开放经济体，同时受地理通道条件约束更强。

## 关键事实与结构特色

- **侨汇重要：** 世界银行数据显示**2024年侨汇约占GDP的4.9%**。
- **"侨汇+服务出口"是核心通道：** IMF分析把侨汇与服务出口列为关键宏观渠道。
- **追赶型小经济体：** IMF对外部风险（贸易伙伴增长、外部环境变化）给出较高权重的风险表述。
- **宏观重点在缓冲与韧性：** 更强调政策缓冲与冲击管理，而非全球金融影响力。

## Core Pillars（金融层）

**侨汇 + 服务/IT类可贸易部门**
以侨民联系与服务出口支撑需求与外部稳定的结构。

## Global Identity（直观层）

**侨民链接的高加索经济体**
以侨民网络与侨汇循环的经济特征为主要外部认知。
