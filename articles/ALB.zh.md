# 阿尔巴尼亚：旅游快速增长、水电依赖与列克本币下的追赶型经济

阿尔巴尼亚是西巴尔干的小型经济体，近年的结构特征非常鲜明：**旅游增长很快**、电力系统高度依赖**水电**，金融体系以**银行为主**，并运行在**本币列克（ALL）**框架下（不是欧元区国家）。从结构上看，阿尔巴尼亚更像一个欧盟邻接的追赶型经济体，外部流入（旅游收入与侨汇）对国内需求与外汇流动性较重要。

## 关键事实与结构特色

- **欧盟路径（制度趋同）：** 阿尔巴尼亚于**2014年6月**获得欧盟候选国地位，并于**2022年7月**开启入盟谈判。
- **货币：** 官方货币是**阿尔巴尼亚列克（ALL）**。
- **旅游带来的外部收入：** 据阿尔巴尼亚旅游部数据（Euronews Albania转述），**2024年约有1170万外国访客**。
- **侨汇仍是结构性支撑：** 基于世界银行口径的估算显示，阿尔巴尼亚**2024年侨汇约占GDP的8.37%**。
- **水电占绝对主导：** UNECE资料指出，阿尔巴尼亚可再生电力装机中**约95%为水电**。

## Core Pillars（金融层）

**旅游与侨汇外部流入 + 水电主导能源结构**
外部流入支撑消费与外汇循环，而电力供给对水文条件（来水）更敏感。

## Global Identity（直观层）

**"新兴"的亚得里亚海旅游目的地**
西巴尔干中快速崛起的旅游市场，在本币体系下运行并沿欧盟候选国路径推进制度趋同。
