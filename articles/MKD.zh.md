# 北马其顿：欧盟候选国的供应链制造节点与侨汇支撑经济

北马其顿是西巴尔干的内陆小国，其经济结构主要由三条线共同塑形：欧盟候选国的制度趋同、外资带动的轻制造/装配型出口、以及侨汇对外汇与消费循环的支撑。它不是金融中心；金融体系更偏银行主导，宏观稳定运行在**本币第纳尔（MKD）**框架下，而非欧元体系。

## 关键事实与结构特色

- **欧盟路径：** 2005 年成为欧盟候选国，2022 年开始入盟谈判。
- **货币：** 使用本币第纳尔（MKD），非欧元区、非"事实欧元化"。
- **侨汇：** 世界银行持续跟踪其侨汇占 GDP 比例，显示侨汇是结构性支撑之一。
- **经济底盘：** 典型的欧盟邻接、外向型小经济体，更多依赖外部市场与供应链链接，而非国内规模。

## Core Pillars（金融层）

**外资驱动的出口制造 + 本币第纳尔宏观金融**
通过欧盟邻接的产业链与投资实现升级，在本币框架下管理通胀、利率与宏观稳定。

## Global Identity（直观层）

**西巴尔干欧盟候选国制造节点**
以欧盟邻接的生产与服务承接、侨汇链接与制度趋同路径为主要外部识别。
