# 格鲁吉亚：黑海通道型经济与侨汇/服务业外部流入

格鲁吉亚的结构角色核心是**通道经济**（黑海—高加索连接）+ **开放型外部平衡**（依赖服务与外部流入支撑）。

## 关键事实与结构特色

- **通道地位：** 里海油气外输的重要基础设施之一——**BTC管道**从阿塞拜疆通往土耳其，途经格鲁吉亚。
- **外部平衡依赖流入：** IMF指出随着侨汇回归常态，经常账户赤字可能扩大。
- **侨汇占比显著：** 世界银行数据显示**2024年侨汇约占GDP的11.9%**。
- **服务出口重要：** 运输/IT等服务对外部收支有重要支撑作用。

## Core Pillars（金融层）

**黑海通道 + 侨汇/服务支撑的外部平衡**
以通道服务与侨汇等流入支撑开放经济的外部融资。

## Global Identity（直观层）

**高加索门户（黑海连接器）**
以"连接里海—黑海—更广市场"的通道属性被识别。
